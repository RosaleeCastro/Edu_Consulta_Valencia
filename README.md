# EduConsulta Valencia

Proyecto académico para consumir una API pública externa y aplicarla a una herramienta útil dentro de una futura plataforma educativa.

## Objetivo

Crear un buscador de centros educativos de la Comunitat Valenciana utilizando datos abiertos oficiales de la Generalitat Valenciana.

## Tecnologías

- HTML5
- CSS3
- JavaScript
- PHP
- API pública externa
- MySQL como mejora opcional

## Funcionalidades iniciales

- Formulario de búsqueda por localidad.
- Filtro por provincia.
- Filtro por régimen.
- Consulta a una API interna en PHP.
- Consumo de datos desde una API pública externa.
- Visualización de resultados en tarjetas.

## Valor añadido

La aplicación no solo muestra datos, sino que los transforma en una consulta clara y útil para una asociación o plataforma educativa.

## Autoría

Proyecto realizado como práctica académica de Desarrollo de Aplicaciones Web.
